ABOUT ETHSTARTER

What is EthStater?

Ethstater is a fully descentralized web application that allows users to get their projects funded by using the power of Ethereum and IPFS

Smart Contracts

The core logic of Ethstater is written in a set of Etthereum Smart Contracts, is completely open source and can be found in our Github.

We decided to do be completely open because we want to be 100% transparent with what our system is doing and allow the comunity to help us to keep improving.
